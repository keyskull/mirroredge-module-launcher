# mmultiplayer (archived monolith)

This tree is the pre-split **mmultiplayer.dll** implementation. It is isolated here and **excluded from the solution build**.

Active development lives in:

- `core/` — plugin shell, menu, IPC, settings
- `engine/` — SDK, gameplay hooks, `MMOD_GetMmultiplayerApi`

See `docs/architecture.md` and `docs/module-manager.md`.
